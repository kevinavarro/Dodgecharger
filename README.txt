# Dodgecharger
A web page that utilizes HTML, CSS and JavaScript.

To Run:
Download .zip file, ofinally open command prompt and change directories until you are in Dodgecharger. Finally, copy and paste this command:

if Google chrome is NOT your default use this command:
browser-sync start --server --browser "Google Chrome" --files "*.html, stylesheets/*.css, scripts/*.js"

if Google chrome is your default use this command:
browser-sync start --server --browser --files "*.html, stylesheets/*.css, scripts/*.js"

The list of files include: index.html, main.js, styles.css, flowmaster.jpg, LED.jpg, otter5.jpg, subwoofer.jpg, tint.jpg, favicon.ico
